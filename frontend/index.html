A decentralised restaurant management system built on Ethereum. Connect your MetaMask wallet to access the full DApp.

Before connecting

1Install MetaMask browser extension
2Add Hardhat Local network (RPC: http://127.0.0.1:8545, Chain ID: 31337)
3Import Account #0 from npx hardhat node
4Make sure the backend API is running on port 3001
